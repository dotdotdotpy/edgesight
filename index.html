edgesight

Tools for catching what your runtime won't.

buildwatcher.py

Watches a build folder. Warns when nothing happens.

ghostenv.py

Scans Python environments for hidden issues, path poisoning, or shadowed imports.

loop-jammer.py

Detects infinite/stuck loops from the inside. Exits when limits are crossed.

stackwhisper.c

Scans live stack memory for pattern leakage. Useful for forensic memory introspection.

stackghost.c

Self-hijacking C utility that embeds a payload in memory and executes it through stack scanning.
